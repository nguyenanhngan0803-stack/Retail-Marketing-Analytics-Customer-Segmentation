# Retail Marketing Analytics & Customer Segmentation

## 1. Project Overview

This project analyzes customer behavior and marketing performance using Python-based analytics.  
The goal is to segment customers, predict spending behavior, evaluate campaign response, and provide data-driven marketing recommendations.

## 2. Business Problem

Marketing teams need to understand which customer groups are more valuable, which behavioral factors influence spending, and how campaign targeting can be improved.

This project answers questions such as:

- Which customers are high-value customers?
- Which factors are most related to customer spending?
- Can customer response or spending behavior be predicted?
- How can customer segments support better campaign decisions?

## 3. Dataset

The dataset includes customer demographics, purchasing behavior, spending information, and campaign response data.

Key analytical features include:

- Income
- Age
- Recency
- Total spending
- Web visits
- Purchase channels
- Campaign response
- Customer segments
